## HTML5 Canvas

### Canvas API

Canvas API는 자바스크립트와 <canvas> 엘리먼트를 이용해 애니메이션, 게임 그래픽, 데이터 시각화, 사진 조작 및 실시간 비디오 처리와 같은 그래픽을 그리기 위완 수단을 제공한다.
